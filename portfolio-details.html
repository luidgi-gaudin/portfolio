BMW Autoshow

Ce projet est un projet réalisé en formation visant à tester nos compétence en HTML, CSS et JS. Le but de ce projet était de faire un CRUD (create, read, update, delete) en JS. Dans ce projet nous retrouvons une page de connection, une page avec un tableau de données modifiable via des Popup. Et sur une autre page nous retrouvons un tableau contenant des données provenant d'une api externe.
