BMW Autoshow

Ce projet est un projet réalisé en formation, visant à tester nos compétences en HTML, CSS et JS. Le but de ce projet était de faire un CRUD (create, read, update, delete) en JS. Dans ce projet, nous retrouvons une page de connexion, une page avec un tableau de données modifiable via des Popups. Et sur une autre page, nous retrouvons un tableau contenant des données provenant d'une API externe.
